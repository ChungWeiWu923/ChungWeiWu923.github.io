Leveraging AI to Identify the KOL

2024.01 - 2024.06

This capstone project is a collaboration between the department of electrical engineering in University of Washington and Genmab, an international biotech company specializing in antibody therapeutics for the treatment of cancer and other serious diseases. This project aims to develop a machine learning powered application to map current KOLs with the highest influence and predict future leaders in medicine field. You could access the final website through this link: http://findkol.com/search/

  • Technologies: Python, NetworkX, Graph Neural Network, PyTorch, Django, RESTful API, MySQL, web scraping, machine learning

This project is aim to develop a front-end application with web-based UI and algorithm in Python based on machine learning (ML) to identify key opinion leaders (KOLs) in the pharmaceutical industry. The KOL in pharmaceutical industry usually have high citations and high number of clinical trials, so those people are the target we are seeking for. This project could be divided into three parts: the core machine learning (ML) algorithm to provide a list of KOLs for certain disease, a back-end system with database to store those lists, and a front-end web page UI for user to search the result. I personally involved in the development of ML part and the back-end part.

For the ML part, we use some web scraping scripts in python with Entraz, an online search system provided by NCBI, to pull down the publications and clinical trials dataset from online database like PubMed and ClinicalTrials.gov. After the data preprocessing process to clean and remove abnormal data such as the work-in-progress trials and store those data in to JSON file, we use NetworkX to construct the citation network or co-trials network. The node in those network represents a researcher or a doctor. Then we calculate the centrality score to rank those node based on the overall influence to that network with the acceleration of using graph neural network constructed with PyTorch. The ranking list of KOL is the output of the stage.

For our back-end implementation, we utilize Django to create RESTful APIs for the front-end and MySQL to store KOL data. Our web application has three main routes:

  • /add: Adds data to the database.
  • /search: Main page displaying prompts and filters. Users can apply filters and submit to retrieve the top 50 KOLs meeting the criteria
  • /info: Displays detailed information for the selected KOL

We have implemented three corresponding APIs using Django. Each API processes GET requests, parses the request parameters, executes SQL queries on the database, and displays the retrieved data on the respective page.

Lidar-based Self Driving Car

2023.09 - 2023.12

This is the final project of a course, The Self Driving Car: Introduction To Ai For Mobile Robots, offered by the University of Washington. The aim of this project is to develop software for running on a MuSHR rally car to win a racing tournament by navigating through the basement of the EE building in the shortest time.

  • Technologies: Python, Nvidia Jetson Nano, ROS, Particle filter algorithm, Lidar, Probability sensing model, A* algorithm

The goal of this project is to develop software for a MuSHR rally car to win a racing tournament by navigating the EE building basement in the shortest time. The car must follow designated path points (marked as the blue one on the floor), maintain a safe distance from certain points (marked as the red one on the floor), and avoiding obstacles in the basement area.

To do so, we provide a preplanned path to the car, and the car would use its lidar sensor to sense the environment and adjust its maneuvers based on the sensing data and the preplanned path. We use the a-star algorithm to find the shortest path, a probability sensor model for processing laser scan data from lidar, the particle filtering algorithm to estimate car's state and the surrounding environment, and PID control for steering the car. The software, written in Python, runs under the ROS framework on an Nvidia Jetson Nano development board.
